== Description ==

This is a simple plugin that outputs a table with user data in the frontend via shortcode.
The table is also displayed on the admin page of the plugin, with an option to export selected items (users) to csv

== Installation ==

###Fast and easy install

#### From the wordpress admin

1. Log into your WordPress site and navigate to the admin area.
2. Go to: **Plugins &gt; Add New**.
3. Enter **"MemberPress Challenge"** in the search field located at the top right corner of the page.
4. Once you find the plugin (that will never happen!!:p) click on **"Install Now"**
5. Select **"Install Now"**
6. After the installation success message, select **"Activate"** and you’re ready to go.

###Alternative installations

**Download a copy of the plugin and save it to your computer.** <br>
Now you can install it using the wordpress admin interface or by manually uploading it.

#### From the wordpress admin

1. Log into your WordPress site and navigate to the admin area.
2. Go to: **Plugins &gt; Add New**.
3. Select **"Upload Plugin"** from the top of the page.
4. Click on **"browse"** and locate the plugin you downloaded on your hard drive.
5. Select **"Install Now"**
6. After the installation success message, select **"Activate"** and you’re ready to go.

#### Manual upload

1. Manually upload the plugin to your wordpress plugins folder.
2. Extract the zip file
3. Navigate to your Wordpress admin area
4. Locate the plugin and select **"Activate"**

## How to use the plugin

The plugin shortcode is [my-mp-challenge] . The are no parameters available.

Add the shortcode in some wordpress content (ex: page, post) and you will see the wonderful table pop up!
